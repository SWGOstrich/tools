Features

  • Open SWG's TRE and TOC files and display their contents
  • Preview all files internally
    • Preview DDS files as an image with zoom, scroll, and save as (BMP, JPG, PNG, TGA) functionality
    • Preview STF files as a table
    • Preview datatable IFF files as a table
    • Preview form IFF files as an ASCII dump
    • Preview PAL files as a color palette
    • Preview ILF files as an ASCII dump
    • Preview WS files as an ASCII dump
    • Preview MP3 and WAV files as playable sounds
    • Preview all other files as an ASCII dump
  • Edit many files internally
    • Edit datatable IFFs as a table with save functionality
    • Edit form IFFs as a node/hex view with copy/cut/paste, insert/overwrite, find/replace, and save functionality
    • Edit ILMs as a node/property view with save functionality
    • Edit PALs as a color palette with add/remove/change color and save functionality
    • Edit STFs as a table with add/remove and save functionalirt
    • Edit WSes as a node/property view with save functionality
    • Edit many other file types using the form IFF editor
  • Export single files, batch files, and batch folders
  • Search by filename within the TOC/TRE contents
  • Sort by name, file type, containing TRE, path, size, or compressed size
  • Filter by containing TRE and/or file type
