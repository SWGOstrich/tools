IFF file extensions

The following file extensions use the Interchange File Format and can be modified in the IFF Editor:

  • ANS
  • APT
  • ASH
  • CDF
  • CEF
  • CMP
  • EFT
  • FLR
  • IFF
  • LAT
  • LAY
  • LMG
  • LOD
  • LSB
  • LTN
  • MGN
  • MKR
  • MSH
  • PLN
  • POB
  • PRT
  • PSH
  • PST
  • SAT
  • SFP
  • SHT
  • SKT
  • SND
  • SPR
  • SWH
  • TRN
  • TRT
