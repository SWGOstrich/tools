Installation instructions

  • Download and install the Microsoft .NET Framework Version 2.0 for your operating system
    • Microsoft .NET Framework Version 2.0 Redistributable Package (x86)
    • Microsoft .NET Framework Version 2.0 Redistributable Package (x64)
  • Download and install the Microsoft DirectX End-User Runtime
  • Download the latest TRE Explorer package
  • Unzip the package to any folder
  • Run TRE Explorer.exe
